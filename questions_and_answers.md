###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A

<i> let is used to declare variable. you can declare a variable using 'let' without assigning an initial value, and then assign a value later.</i>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: D

<i>step-1)condition true --> count = 0 + 1  // ans: 1
  step-2)condition true --> count = 1 + 1  // ans: 2
  step-3)condition true --> count = 2 + 1  // ans: 3
  step-4)condition true --> count = 2 + 1  // ans: 4
</i>

</p>
</details>
